# 🪟 Windows 11 Web Simulator & Prank Tool

![Build](https://img.shields.io/badge/Build-Web%20Edition-brightgreen?style=for-the-badge)

Bu proje, modern **Windows 11** arayüzünün saf HTML, CSS ve JavaScript kullanılarak oluşturulmuş yüksek sadakatli bir simülasyonudur. Sadece bir görsel kopya değil, aynı zamanda etkileşimli pencereler ve gizli şaka (prank) mekanizmaları içeren bir deneyim sunar.

## 🚀 Canlı Demo
[Buraya GitHub Pages Linkini Yapıştır](https://n98540638-beep.github.io/win11/)

## ✨ Özellikler

* **🎯 Pixel-Perfect UI:** Windows 11'in ikonik ortalanmış görev çubuğu ve Mica (cam) efekti tasarımı.
* **🗔 Pencere Yönetimi:** Sürüklenebilir, odaklanabilir ve kapatılabilir dinamik pencere sistemi.
* **📂 Başlat Menüsü:** Windows 11 stili animasyonlu açılış ve uygulama listesi.
* **🕒 Canlı Sistem:** Gerçek zamanlı saat ve sistem tepsisi simgeleri.
* **😈 Prank Mode (BSOD):** Belirli etkileşimlerle tetiklenen gerçekçi "Mavi Ekran" (Blue Screen of Death) simülasyonu.
* **⚠️ Özel 404:** Yanlış yönlendirmelerde otomatik devreye giren BSOD hata sayfası.

## 🛠️ Kullanılan Teknolojiler

* **HTML5:** Semantik yapı ve uygulama iskeleti.
* **CSS3:** Glassmorphism, Flexbox/Grid ve gelişmiş animasyonlar.
* **JavaScript (Vanilla):** DOM manipülasyonu, sürükle-bırak mantığı ve sistem işlevleri.
